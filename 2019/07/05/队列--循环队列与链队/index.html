注意：队列也是线性表，其特殊性在于有特殊的运算规则。即：队结构只能在一端进行插入，该操作端称为队尾，另一端删除元素，该操作端称为队头。按照“先进先出”（First In First Out,FIFO）原则处理数据节点。

1.循环队列

之所以用循环对列，就是了为了提高利用率。要不然每删除一个元素，对头就空了一个还用不了了。
实现循环，就用“模”运算：
队空的条件：Q.front == Q.rear
堆满条件：（Q.rear+1）%Max == Q.front
